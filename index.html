AK-HUB Packages

About Bandicoot


Bandicoot is a terminal emulator application for Android OS which can be extended by packages of ported common GNU/Linux utilities.

Home page: https://AK-HUB.com

Github: https://github.com/AK-HUB-S-MASTER
